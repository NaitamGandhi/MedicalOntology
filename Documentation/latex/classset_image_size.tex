\doxysubsection{Detailed Description}
This script is used to resize and relocate the background image (this black translucent background) when the nodenemu is open. 

It resizes it for the transition from \char`\"{}\+Nodemenu\+Start\char`\"{} to \char`\"{}\+Nodemene\+Panel\char`\"{} when the \char`\"{}\+View Details\char`\"{} button is clicked.
