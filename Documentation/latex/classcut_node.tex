\doxysubsection{Detailed Description}
This script adds node-\/detaching functionality. 

This script is used to add the functionality of the \char`\"{}\+Cut Node\char`\"{} button. The purpose of this button is to give the user the ability to un-\/link the link of the node from the rest of the Ontology or its lineage, for it to be then attached elsewhere.
